Projet de Programmmation Encadré : Mini-Projet

Présentation

Ce projet s'inclut dans le cadre de l'UE "L7TI005 - Projet de Programmation Encadré", faisant partie du cursus du Master PluriTAL. Il consiste en une préparation pour le travail de groupe qui conclut le semestre.

Pour réaliser ce projet, nous avons eu recours à une liste d'URLs fournie par les enseignants. Afin de faciliter la récupération des données

Tout au long de ce projet, nous avons pu acquérir les compétences nécéssaires à l'extraction de données et avons mis à profit les outils à notre disposition pour obtenir un résultat satisfaisant.

Processus

L'élaboration du Mini-Projet s'est déroulée du 22/10/25 au 17/11/25, s'étalant sur trois semaines. en excluant la semaine de la Toussaint. Le processus a suivi les étapes suivantes :

  1. Écriture d'un script bash qui récupère certaines informations (le code HTTP, le type d'encodage et le nombre de mots) à partir d'une liste d'URLs données.
  2. Inscription des données récoltées lors de l'étape I. dans un fichier tabulaire de type .tsv, où les données sont séparées par des tabulations.
  3. Réadaptation du script afin que la sortie soit sous format .html
  4. Personnalisation de la sortie .html en utilisant la bibliothèque CSS "Bulma"
  5. Création d'une page GitHub résumant le Mini-Projet

Méthodologie

Pour récolter les données sur les différents sites, nous avons utilisé, au sein de notre script bash, les commandes suivantes :

  • curl : Récupère les données dans le code HTML d'une URL donnée
    • -i : Inclut les entêtes
    • -L : Permet les redirections si l'URL n'est plus d'actuaité
  • grep : Sélectionne un patron spécifique pour ne récupérer que ce qui nous intéresse
    • -P : Interprète les patrons en tant qu'expressions régulières compatibles avec Perl
    • -o : Ne reconnait que le patron donné
  • lynx : Sert de navigateur web au sein du terminal
    • -dump : Renvoie le texte brut de la page web
    • -nolist : Supprime la liste d'URLs qui apparait par défaut à la fin de la page
  • tr : Permet de traduire ou supprimer un caractère
    • -d : Permet de définir les caractéres à supprimer
  • cut : Retire une section dans une ligne de texte
    • -d : Permet de définir un séparateur, celui par défaut étant une tabulation
